## Finding Limits - Homework Question Sample

### Description
A simple Streamlit application that helps students practice evaluating limits involving square roots and rational expressions.

### Link
https://findinglimits-practice.streamlit.app/

#### Notes:
- If providing a repeating decimal of a fraction, include at least 9 digits after the decimal point (ex: 1/3 = 0.333333333)
- Only 25 characters are allowed for the answer box
- Click "Try Again" to randomize the limit in the question prompt
- Click "Solution" to view calculation steps
